# Streaming Platform Logos

Upload logo platform streaming di folder ini dengan format:

## Format File:
- **Format**: PNG dengan background transparan
- **Ukuran**: 200x200px atau 400x400px (square)
- **Nama file**: lowercase, tanpa spasi

## Cara Upload:
1. Siapkan logo dalam format PNG
2. Rename sesuai nama platform (lowercase)
3. Upload ke folder `public/logos/`
4. Logo akan otomatis muncul di halaman release detail

## Fallback:
Jika logo tidak tersedia, sistem akan menampilkan emoji sebagai fallback.
